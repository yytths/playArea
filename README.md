# お遊技場

## なにこれ

Vue を使ってお遊技場作ってます。
ゆったり遊び増やしていきたいな。

## できること
* リバーシ

## 始め方
* `npm run dev` で実行。
* `localhost:8080` にアクセス

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
